COLORES

Tenemos distintos tipos de colores:

  1. RGB

    2. El más básico, con 256 combinaciones posibles para cada color Red Green Blue (Un total de 16.777.216 combinaciones posibles)

  2. RGBA

    3. Canal A o Alfa que define la opacidad o transparencia del RGB

  3. HSL
  4. HSLA
  5. HEXADECIMAL

Cada una tiene sus ventajas y formas diferentes
